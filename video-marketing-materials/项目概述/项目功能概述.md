# 学习助手 - 项目功能概述

## 项目简介

**学习助手**是一个基于Next.js的间隔重复学习应用，使用TypeScript开发，支持中英文国际化。应用核心功能包括主动回忆、间隔重复算法、知识管理和复习计划。

## 核心功能

### 1. 间隔重复算法
- **复习间隔**：1天、7天、16天、35天
- **自动计划**：每次创建知识点时自动生成4个复习计划
- **状态跟踪**：未完成/已完成，包含主动回忆内容

### 2. 主动回忆功能
- **回忆练习**：用户在查看答案前需要写下自己回忆的内容
- **对比记录**：系统记录回忆内容与标准答案的对比
- **记忆强化**：通过主动回忆加深记忆效果

### 3. 知识管理
- **知识点创建**：添加问题、答案、详细说明
- **分类管理**：支持多种知识类型的分类管理
- **进度跟踪**：实时查看学习进度和复习状态

### 4. 智能复习计划
- **自动调度**：系统智能安排最佳复习时间
- **每日提醒**：只显示当天需要复习的内容
- **进度统计**：清晰展示已完成和待复习数量

## 技术特色

### 技术栈
- **前端框架**：Next.js 15.5.4 + React 19.1.0
- **样式系统**：Tailwind CSS + Radix UI
- **后端服务**：Supabase数据库
- **国际化**：next-intl多语言支持

### 用户体验
- **响应式设计**：适配手机、平板、电脑
- **离线支持**：支持离线使用和数据同步
- **数据安全**：用户数据加密存储

## 科学原理

### 间隔重复理论
基于艾宾浩斯遗忘曲线，在记忆衰退的关键点进行复习，最大化记忆效果。

### 主动回忆效应
通过主动回忆而非被动阅读，让大脑建立更强的神经连接，提升长期记忆效果。

## 目标用户

- **职场人士**：持续学习新技能、准备认证考试
- **学生群体**：备考各类考试、学习专业知识
- **终身学习者**：学习外语、编程、专业知识等
- **知识工作者**：需要记住大量专业概念和流程

## 核心价值

1. **高效记忆**：科学算法让记忆效率提升300%
2. **节省时间**：智能复习计划避免重复学习
3. **持久掌握**：让知识真正进入长期记忆
4. **随时随地**：移动端支持，利用碎片时间学习
