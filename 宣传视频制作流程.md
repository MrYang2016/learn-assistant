# Learn Assistant 宣传视频制作流程文档

## 📋 项目概述

**Learn Assistant** 是一款基于间隔重复（Spaced Repetition）和主动回忆（Active Recall）科学学习法的智能学习助手应用。本文档旨在指导宣传视频的制作流程。

---

## 🎯 视频目标

### 核心目标
- 向目标用户展示 Learn Assistant 的核心价值和使用方法
- 突出科学学习法（间隔重复 + 主动回忆）的优势
- 展示应用的易用性和智能化特性
- 吸引用户注册和使用

### 目标受众
- **主要受众**：学生、职场人士、终身学习者
- **次要受众**：教育工作者、学习社群管理者
- **痛点**：记忆遗忘、学习效率低、缺乏系统化复习方法

---

## 📐 视频结构规划

### 视频时长建议
- **完整版**：2-3 分钟（适合官网、社交媒体）
- **精简版**：30-60 秒（适合短视频平台）
- **详细演示版**：5-8 分钟（适合产品介绍页、教程）

### 视频结构（完整版）

#### 第一部分：开场（0:00-0:15）
- **视觉**：简洁的 Logo 动画 + 应用名称
- **文案**："还在为记忆遗忘而烦恼？Learn Assistant 帮你科学记忆"
- **音效**：轻快的背景音乐

#### 第二部分：痛点引入（0:15-0:30）
- **视觉**：展示学习场景（学生/职场人士学习画面）
- **文案**：
  - "学了就忘？复习没有计划？"
  - "科学证明：间隔重复 + 主动回忆 = 高效记忆"
- **数据展示**：记忆曲线图表动画

#### 第三部分：核心功能演示（0:30-1:30）
**3.1 知识点管理（0:30-0:50）**
- **操作演示**：
  - 登录/注册界面
  - 添加知识点（问题 + 答案）
  - 知识点列表展示
- **旁白**："轻松添加知识点，系统自动安排复习计划"

**3.2 智能复习计划（0:50-1:10）**
- **操作演示**：
  - 切换到"今日复习"标签
  - 显示待复习数量
  - 展示复习卡片界面
- **旁白**："基于科学算法，自动安排 1天、7天、16天、35天的复习计划"

**3.3 主动回忆功能（1:10-1:30）**
- **操作演示**：
  - 查看复习问题
  - 输入主动回忆内容
  - 显示标准答案对比
  - 完成复习
- **旁白**："主动回忆，加深记忆，让知识真正内化"

#### 第四部分：高级功能（1:30-2:00）
**4.1 MCP API 集成（1:30-1:45）**
- **操作演示**：
  - API Key 管理界面
  - MCP 设置说明
- **旁白**："支持 MCP 协议，可与 AI 助手无缝集成"

**4.2 多语言支持（1:45-2:00）**
- **操作演示**：
  - 语言切换（中英文）
- **旁白**："支持中英文，国际化学习体验"

#### 第五部分：价值总结（2:00-2:15）
- **视觉**：关键数据展示
  - "科学算法"
  - "自动规划"
  - "主动回忆"
  - "高效记忆"
- **文案**："Learn Assistant，让学习更科学，让记忆更持久"

#### 第六部分：行动号召（2:15-2:30）
- **视觉**：应用下载/访问链接
- **文案**："立即开始，开启你的高效学习之旅"
- **CTA**：显示网站链接、二维码

## 🎬 拍摄/制作流程

### 阶段一：前期准备（1-2天）

#### 1.1 素材准备清单
- [ ] **应用截图/录屏**
  - 登录/注册界面
  - 添加知识点界面
  - 知识点列表
  - 今日复习界面
  - 复习卡片（问题、回忆输入、答案展示）
  - API Key 管理界面
  - MCP 设置界面
  - 语言切换界面

- [ ] **设计素材**
  - Logo 动画（AE/After Effects）
  - 图标素材（记忆曲线、大脑、书本等）
  - 背景音乐（轻快、专业）
  - 音效（点击、完成、提示音）

- [ ] **文案脚本**
  - 完整旁白脚本（中英文版本）
  - 字幕文本
  - 关键数据/卖点文案

#### 1.2 技术准备
- [ ] 准备测试账号（包含示例知识点和复习数据）
- [ ] 录制环境设置（屏幕录制工具：OBS、QuickTime、Camtasia）
- [ ] 应用界面美化（确保 UI 处于最佳状态）
- [ ] 准备演示数据（至少 5-10 个知识点，部分处于待复习状态）

### 阶段二：录制阶段（1天）

#### 2.1 屏幕录制
**录制工具推荐**：
- macOS: QuickTime Player / ScreenFlow / OBS
- Windows: OBS Studio / Camtasia
- 在线工具: Loom / Screencastify

**录制设置**：
- 分辨率：1920x1080 (1080p) 或更高
- 帧率：30fps 或 60fps
- 音频：清晰无杂音
- 鼠标指针：显示（可添加高亮效果）

**录制内容**：
1. **完整用户流程录制**（5-8分钟原始素材）
   - 从登录到完成一次复习的完整流程
   - 包含所有核心功能操作
   - 多角度录制（不同功能模块）

2. **关键操作特写**（每个功能单独录制）
   - 添加知识点（慢动作，突出操作步骤）
   - 查看复习列表（展示数据统计）
   - 完成复习（展示主动回忆流程）
   - API 设置（展示技术特性）

#### 2.2 旁白录制
- **设备**：专业麦克风或高质量录音设备
- **环境**：安静、无回声
- **格式**：WAV 或高质量 MP3
- **内容**：按照脚本逐段录制，预留停顿时间

### 阶段三：后期制作（2-3天）

#### 3.1 视频剪辑
**工具推荐**：
- 专业：Adobe Premiere Pro / Final Cut Pro
- 中级：DaVinci Resolve / Filmora
- 简单：iMovie / 剪映

**剪辑步骤**：
1. **素材整理**
   - 导入所有录屏素材
   - 导入旁白音频
   - 导入背景音乐和音效
   - 导入设计素材（Logo、图标等）

2. **时间轴搭建**
   - 按照视频结构规划时间轴
   - 放置背景音乐轨道
   - 放置旁白轨道
   - 放置视频素材轨道

3. **画面剪辑**
   - 裁剪录屏素材，保留精华部分
   - 添加转场效果（淡入淡出、滑动等）
   - 添加关键操作的高亮标注（箭头、圆圈、文字说明）
   - 添加数据展示动画（复习数量、时间线等）

4. **音频处理**
   - 调整旁白音量
   - 背景音乐淡入淡出
   - 添加音效（点击声、完成提示音）
   - 降噪处理（如需要）

5. **字幕添加**
   - 中文字幕（完整版）
   - 英文字幕（国际化版本）
   - 关键信息标注（功能名称、数据等）

6. **特效添加**
   - Logo 动画（开场）
   - 数据可视化动画（记忆曲线、统计图表）
   - 界面高亮效果（鼠标点击、区域高亮）
   - 文字动画（关键卖点文字）

#### 3.2 多版本制作
- **完整版**（2-3分钟）：包含所有功能演示
- **精简版**（30-60秒）：核心功能快速展示
- **详细演示版**（5-8分钟）：完整操作流程，适合教程

### 阶段四：审核与优化（1天）

#### 4.1 内容审核
- [ ] 功能演示是否准确
- [ ] 文案是否清晰易懂
- [ ] 画面是否流畅
- [ ] 音频是否清晰
- [ ] 字幕是否正确

#### 4.2 技术审核
- [ ] 视频分辨率符合要求
- [ ] 音频同步无误
- [ ] 文件大小适中（便于传播）
- [ ] 格式兼容性（MP4/H.264）

#### 4.3 优化调整
- 根据反馈调整节奏
- 优化关键画面停留时间
- 调整背景音乐音量
- 完善字幕和标注

## 🎨 视觉设计规范

### 色彩方案
- **主色调**：应用主题色（参考应用 UI）
- **强调色**：用于高亮关键操作和重要信息
- **背景色**：简洁、不抢夺内容焦点

### 字体规范
- **标题**：粗体、易读（如：思源黑体、PingFang SC）
- **正文**：清晰、专业（如：苹方、Helvetica）
- **字幕**：清晰、对比度高（白色/黄色，带描边）

### 动画风格
- **转场**：流畅、自然（淡入淡出、滑动）
- **标注**：简洁、不干扰（箭头、圆圈、高亮框）
- **数据展示**：动态、直观（数字递增、图表动画）

## 📊 关键数据展示

## 🛠️ 技术规格

### 视频输出规格
- **分辨率**：
  - 主版本：1920x1080 (1080p)
  - 社交媒体：1080x1920 (竖屏，适合抖音、小红书)
  - 4K 版本：3840x2160 (可选，用于官网)
- **帧率**：30fps 或 60fps
- **格式**：MP4 (H.264)
- **码率**：8-15 Mbps（1080p）
- **音频**：立体声，48kHz，192kbps

### 文件命名规范
```
learn-assistant-promo-[版本]-[语言]-[分辨率].mp4
示例：
- learn-assistant-promo-full-zh-1080p.mp4
- learn-assistant-promo-short-zh-1080p.mp4
- learn-assistant-promo-full-en-1080p.mp4
```
